## セットアップ

1. Gitをインストール: https://git-scm.com/download/win
2. GitHub CLIをインストール: https://cli.github.com/
3. 以下のコマンドを実行:

```bash
git init
git add .
git commit -m "Initial commit"
gh repo create hp --public --source=. --remote=origin --push
```
